# App Starter Blank

This Phase 3 fixture is the app's first-run ROM. It intentionally shows a
blank Genesis screen so the right pane can prove that a ROM is running before
the user asks the agent to add sprites or music.

Build from the repo root:

```sh
scripts/build-sgdk.sh examples/app-starter-blank
```

Expected output:

```text
Built ROM: .../examples/app-starter-blank/out/rom.bin
```
