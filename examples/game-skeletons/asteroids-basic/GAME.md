# Asteroids Starter

## Objective

Clear the asteroid field before the ship loses all three lives.

## Controls

- Left / Right: turn
- Up: thrust one step
- A: fire
- C: pause or resume
- Start: begin from the title screen or restart the current run

## Recovery

Restart action: Start. It resets score, lives, ship position, bullets, and every asteroid.

## Status

Deterministic starter scaffold. It remains a Prototype until Drive16 completes independent screen, input, audio, game-over, restart, and visual-coherence review.
