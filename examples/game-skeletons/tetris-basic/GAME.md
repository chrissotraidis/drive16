# Tetris Starter

## Objective

Move and rotate falling pieces, complete lines, and keep the stack below the top of the playfield.

## Controls

- Left / Right: move
- Down: fast drop
- Up or A: rotate
- C: pause or resume
- Start: restart the board, score, and line count

## Recovery

Start action: Start. The game waits on a title/objective screen before pieces begin falling.

Restart action: Start.

## Status

Deterministic starter scaffold. It remains a Prototype until independent semantic and visual review passes.
