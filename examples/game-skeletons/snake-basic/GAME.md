# Snake Starter

## Objective

Guide the snake to food, grow, and avoid the border and the snake's own body.

## Controls

- D-pad: change direction
- C: pause or resume
- Start: restart after a collision or reset the current run

## Recovery

Start action: Start. The game waits on a title/objective screen before the first run.

Restart action: Start. It restores the initial snake, score, direction, and food state.

## Status

Deterministic starter scaffold. It remains a Prototype until independent semantic and visual review passes.
