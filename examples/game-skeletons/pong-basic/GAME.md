# Pong Starter

## Objective

Move the left paddle to return the ball and reach nine points before the CPU.

## Controls

- Up / Down: move the player paddle
- C: pause or resume
- Start: reset the score, paddles, and serve

## Recovery

Start action: Start. The game waits on a title/objective screen before serving.

Restart action: Start.

## Status

Deterministic starter scaffold. It remains a Prototype until independent semantic and visual review passes.
