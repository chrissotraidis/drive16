# Game Notes

## Concept

**Skyline Intercept** is a compact Missile Command-style defense game for the
Sega Genesis. The player moves a radar cursor, detonates interceptors around
incoming missiles, protects four city clusters, and survives escalating waves.

## Controls

- D-pad: move the targeting cursor.
- A or B: detonate an interceptor at the cursor.
- Start: pause or resume; rebuild the cities after game over.
- C: reset the current defense run.

## Expected first screen

The ROM opens on a safe title/instruction screen. It states the objective,
shows aim/fire/restart controls, and waits for Start before any danger begins.
Gameplay uses a framed cyan radar, four readable skyline silhouettes, a small
crosshair, a persistent control legend, and a four-second opening grace period.
Music begins in the ROM, while Drive16 keeps browser playback muted until the
user enables sound.

## Genre evidence required before PASS

The screen must show incoming missiles moving toward cities, D-pad cursor
movement, A/B explosions that can destroy missiles, city loss on impact, score
changes, Start pause/resume, C reset, and non-silent audio evidence.
