# Phase 0 Assets

Original Drive16 validation assets generated by
`scripts/generate-phase0-assets.py`.

- `player.png`: 32x32 indexed-color sprite with transparent palette index 0.
- `loop.vgm`: one-second PSG-only VGM loop for SGDK XGM conversion.

These are intentionally tiny so Phase 0 validates SGDK resource wiring without
depending on commercial art, copied game music, or generated AI asset pipelines.
