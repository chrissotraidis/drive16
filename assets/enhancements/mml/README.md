# Drive16 MML Enhancement Assets

Phase 4 generated music assets live here. They are optional enhancement
contracts and are not part of the CORE bundled-asset path.

Files:

- `fm-presets.mml`: original starter YM2612 FM voices for `ctrmml`.
- `manifest.json`: stable preset IDs, instrument numbers, roles, and short
  sample phrases for validation and later prompt context.

Use these presets when asking the model to write Megadrive MML. The next Phase
4 slices add the MML reference to RAG and wire the prompt path.
